# pinina的核心属性以及插件的原理

## $state

可以将store全部的换掉，可以获取全部的store的状态，也可以替换state的状态

```js
const handleClick = () => {
  store.$state = {count:100}
}

Object.defineProperty(store,'$state',{
    get:()=> pinia.state.value[$id],
    set:(state)=> $patch(($state)=> Object.assign($state,state))
})
```
